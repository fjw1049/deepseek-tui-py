🔐 PART 1: Secrets 密钥管理系统

1. API Key 解析优先级（瀑布模型）

优先级 1

环境变量 (Environment Variables)

DEEPSEEK_API_KEY, OPENAI_API_KEY, NVIDIA_NIM_API_KEY, OPENROUTER_API_KEY

最高优先级 推荐

↓
优先级 2

操作系统密钥环 (OS Keyring)

macOS Keychain / Windows Credential Manager / Linux Secret Service

中等优先级 安全

↓
优先级 3

配置文件 (config.toml)

providers.<provider>.api_key

最低优先级 不推荐 明文存储

2. Keyring 密钥环集成

服务配置

  • 服务名称: "deepseek-tui"
  • 账户键结构: "<provider>_api_key"
  • 示例: "deepseek_api_key"
  • 示例: "openai_api_key"

Keyring 操作

  • get_password(service, account) → Result<String>
  • set_password(service, account, password) → Result<()>
  • delete_password(service, account) → Result<()>

3. 各提供商密钥配置

提供商 环境变量 Keyring 账户 Config 路径
DeepSeek DEEPSEEK_API_KEY deepseek_api_key providers.deepseek.api_key
OpenAI OPENAI_API_KEY openai_api_key providers.openai.api_key
NVIDIA NIM NVIDIA_NIM_API_KEY nvidia_nim_api_key providers.nvidia_nim.api_key
OpenRouter OPENROUTER_API_KEY openrouter_api_key providers.openrouter.api_key

4. 安全特性

🔒 存储安全

  • 密钥永不写入磁盘（除 config.toml 回退）
  • Keyring 访问需要 OS 认证（Touch ID / 密码）
  • 内存中密钥使用后立即清除

🛡️ 日志安全

  • 日志中密钥自动脱敏（替换为 "***"）
  • 错误消息不包含完整密钥
  • 调试模式下也不显示明文密钥

⚠️ 配置文件警告

  • 检测到 config.toml 中有密钥时发出警告
  • 建议迁移到环境变量或 Keyring
  • 启动时显示安全提示

// Rust 结构定义
pub struct Secrets {
    providers: HashMap<String, String>,  // provider → api_key
}

pub enum SecretsSource {
    Env,      // 环境变量
    Keyring,  // OS 密钥环
    Config,   // config.toml
}

impl Secrets {
    pub fn get(&self, provider: &str) -> Option<&String>
    pub fn set(&mut self, provider: &str, key: String, source: SecretsSource)
    pub fn redact(&self, text: &str) -> String  // 脱敏处理
}

📦 Agent 核心结构体 
// Rust 结构定义
pub struct AgentRegistry {
    models: HashMap<String, ModelInfo>,
}

pub struct ModelInfo {
    pub id: String,
    pub provider: String,
    pub max_context: usize,
    pub capabilities: ModelCapabilities,
    pub pricing: Option<PricingInfo>,
}

pub struct ModelCapabilities {
    pub streaming: bool,
    pub tools: bool,
    pub json_mode: bool,
    pub thinking: bool,
}

impl AgentRegistry {
    pub fn resolve(&self, name: &str) -> Result<ModelInfo>
    pub fn get_provider(&self, model: &str) -> Option<&str>
    pub fn get_capabilities(&self, model: &str) -> Option<&ModelCapabilities>
}

⚠️ 错误处理

Secrets 错误

  • 缺失 API Key: 提示用户设置或退出
  • Keyring 访问被拒: 回退到 Env 或 Config
  • 无效密钥格式: 验证错误，显示格式要求
  • 权限不足: 提示需要 OS 认证

Agent 错误

  • 未知模型: 建议相似模型列表
  • 提供商不匹配: 显示正确的提供商
  • 能力不支持: 优雅降级或提示切换模型
  • 别名冲突: 显示规范模型 ID
