🔧 ToolCapability 能力枚举

ReadOnly

只读操作，从不修改状态。例如：读取文件、查询数据、列出目录。

WritesFiles

写入文件系统。例如：创建文件、修改文件、删除文件。

ExecutesCode

执行任意 Shell 命令。例如：运行脚本、编译代码、系统命令。

Network

发起网络请求。例如：HTTP 请求、API 调用、下载文件。

Sandboxable

可在沙箱环境中运行。隔离执行，限制权限。

RequiresApproval

执行前需要用户批准。高风险操作，需要明确授权。

🔤 工具名称编码/解码

编码规则 (to_api_tool_name)
// 编码规则： // 1. 字母数字 + 下划线 → 直接通过 // 2. 连字符 (-) → 转换为双连字符 (--) // 3. 其他字符 → 转换为 -x<hex>- 示例： "web.run" → "web-x00002E-run" "mcp-server" → "mcp--server" "file:read" → "file-x00003A-read" "tool_name" → "tool_name" (无需编码)
解码规则 (from_api_tool_name)
// 解码规则（两遍解析）： // 第一遍：基于分隔符的解码 // - 识别 -x<hex>- 模式并解码 // - 将 -- 转换回 - // 第二遍：裸十六进制解码 // - 处理没有分隔符的十六进制序列 示例： "web-x00002E-run" → "web.run" "mcp--server" → "mcp-server" "file-x00003A-read" → "file:read"

⚠️ 错误处理

InvalidInput

验证失败，输入参数不符合要求。

MissingField

缺少必需字段，无法执行工具。

PathEscape

路径逃逸，尝试访问工作空间外的文件。

ExecutionFailed

执行失败，工具运行时发生错误。

Timeout

操作超时，执行时间超过限制。

NotAvailable

工具不可用，未找到对应的处理器。

PermissionDenied

权限拒绝，授权失败或用户拒绝批准。

🔌 ToolHandler 接口

ToolHandler Trait
#[async_trait] trait ToolHandler: Send + Sync { // 工具名称 fn name(&self) -> &str; // 工具能力 fn capabilities(&self) -> Vec<ToolCapability>; // 是否为修改操作 fn is_mutating(&self) -> bool { self.capabilities().iter().any(|c| matches!( c, ToolCapability::WritesFiles | ToolCapability::ExecutesCode )) } // 处理工具调用 async fn handle(&self, invocation: ToolInvocation) -> Result<ToolResult>; // 验证输入 fn validate(&self, payload: &ToolPayload) -> Result<()> { // 默认实现：检查必需字段 Ok(()) } }

🔄 完整执行流程图

从 Engine 到 Tool 的完整调用链
Engine::execute_op() ↓ Engine::process_tool_calls() ↓ ToolRegistry::dispatch(tool_call, allow_mutating) ↓ 1. 查找 Handler: registry.handlers.get(tool_name) ↓ 2. 查找 Spec: registry.specs.get(tool_name) ↓ 3. 验证 Payload Kind: payload.kind() == handler.kind() ↓ 4. 检查 Mutating: if handler.is_mutating() && !allow_mutating → Error ↓ 5. 创建 ToolInvocation: ToolInvocation::new(call_id, tool_name, payload, source) ↓ 6. 获取执行锁: - 如果 parallel: runtime.lock.read().await - 如果 serial: runtime.lock.write().await ↓ 7. 执行 Handler: tokio::time::timeout(timeout, handler.handle(invocation)) ↓ 8. 返回结果: - 成功: ToolOutput { tool_use_id, content, is_error: false, metadata } - 失败: FunctionCallError { tool_use_id, error_message }
